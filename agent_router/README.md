# Agent Router System

This folder is the operational documentation system for AI assistants.

- Start at `AGENTS.md` (repo root), then read `docs/agent_router/00_AUTHORITY.md`.
- Load only the module(s) needed for the current intent.
- Keep `docs/shared/*` and `docs/projects/veterinary-medical-records/*` as human reference documents.
- If a rule changes, update its owner in `docs/agent_router/00_RULES_INDEX.md`.
